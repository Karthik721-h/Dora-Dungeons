You are a senior game UI/UX designer and frontend architect.

You are redesigning the frontend of an existing audio-first RPG called "Dora Dungeons".

The current system works correctly (game engine, API, voice, audio), but the UI looks like a developer tool. Your task is to transform it into a PROFESSIONAL, IMMERSIVE, GAME-QUALITY INTERFACE without breaking any logic.

---

## CORE RULE (CRITICAL)

* DO NOT change or break any game logic
* DO NOT modify the game engine
* ONLY refactor UI, layout, styling, and presentation layer
* All existing functionality must still work

---

## GOAL

Create a cinematic, immersive, audio-first interface inspired by:

* Dark Souls (minimal HUD)
* Diablo IV (dark fantasy aesthetic)
* Shades of Doom (audio-first interaction)

The UI must feel like a GAME, not a dashboard.

---

## PART 1 — CINEMATIC INTRO SYSTEM

Create a new IntroScene component:

Features:

1. Fullscreen cinematic intro screen

   * Dark background
   * Centered logo (DORA DUNGEONS)
   * Subtle animated glow or fog effect

2. Video support:

   * Auto-play intro video (local or placeholder)
   * Muted by default
   * Optional subtitles overlay

3. Controls:

   * "Skip Intro" button
   * Voice command support: "skip intro"

4. Transition:

   * Fade to black (1–2s)
   * Then fade into game UI

5. Optional narration (TTS):

   * "You descend into darkness..."
   * Must integrate with AudioManager

6. Must be isolated:

   * IntroScene.tsx
   * Controlled by App state (intro → game)

## PART 2 — COMPLETE UI LAYOUT REFACTOR

Replace current layout with a GAME HUD structure:

NEW LAYOUT:

TOP BAR:

* Minimal info: game state (Exploring / Combat), audio status, session ID

MAIN AREA (FOCUS):

* Large narration panel (centered, dominant)

BOTTOM SECTION:
