Implement the ShopService logic for "Dora Dungeons".

* Buying weapons
* Selling items
* Upgrading armor

---

## FUNCTIONS TO IMPLEMENT

1. getAvailableWeapons()

Return a static list of 6 weapons:

* Ashbringer (10 gold)
* Keyblade (20 gold)
* Leviathan Axe (45 gold)
* Gjallarhorn (50 gold)
* Buster Sword (60 gold)
* Gunblade (100 gold)

Each weapon must include:
id, name, description, price

---

2. buyWeapon(player, weaponId)

Logic:

* Find weapon from list
* If player.gold < price → return failure
* Else:

  * Deduct gold
  * Add weapon to player.weapons

Return:
{
success: boolean,
message: string,
updatedPlayer
}

---

3. sellItem(player, itemId)

Logic:

* Find item in player.inventory
* If not found → return failure
* Else:

  * Remove item
  * Add item.value to player.gold

Return same structure

---

4. upgradeArmor(player, armorId)

Rules:

* Max level = 3
* If armor not found → fail
* If already level 3 → fail

Upgrade cost:

* Level 1 → 2 = 20 gold
* Level 2 → 3 = 30 gold

Logic:

* Check gold
* Deduct gold
* Increase armor level

Return same structure

---

Create helpers:

* findItemById()
