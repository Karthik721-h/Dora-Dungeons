## GOAL

1. Via navbar dropdown (manual)

voicePreference: "female" | "male"

* Persist in localStorage
* Load on app start

1. Detect available voices using speechSynthesis.getVoices()

2. Select voices:

* Female → prefer voice with "female" or higher pitch
* Male → prefer voice with "male" or lower pitch

3. Apply selected voice dynamically before speaking

4. If exact match not found:

* Fallback to closest match

* Icon or label: "Voice"

* Female (default)
* Male

* Update voicePreference
* Save to localStorage
* Immediately apply to TTS

Speak confirmation:

If switched to male:
"Voice changed to male narrator."

If switched to female:
"Voice changed to female narrator."
